## Getting started:
1. Install Docker
2. clone repo and cd into it
```bash
    git clone https://github.com/jonasschultemattler/wce
    cd wce
    ```
3. Build the Docker image:
```bash
    docker build -f docker/Dockerfile -t wce .
    ```
4. Download datasets...
5. Setup python
6. Setup gurobi
7. Run exact solver
